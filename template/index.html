The system takes image files of X-Ray scans of the knee, then classifies them according to the Kellgren-Lawrence Grading System using Deep Learning, specifically Convolutional Neural Networks (CNNs).

This system uses the VGG-16 model with optimizer SGD, learning rate 0.0001, epoch size 30, batch size 8. It has an F1-score of 28.34%.

Instructions: User must upload a knee X-Ray image (JPG, JPEG, and PNG only).

Knee osteoarthritis (KOA) is the most prevalent type of osteoarthritis. With no existing cure, treatment is primarily focused on managing symptoms. Therefore, the timely detection of KOA is crucial for effective treatment planning and symptom management.

Kellgren-Lawrence Grading Scale

Grade 0: No radiographic OA present.

Grade 1: Doubtful joint space narrowing and possible osteophyte lipping.

Grade 2: Definite osteophytes and possible joint space narrowing (JSN) on AP radiograph.

Grade 3: Multiple osteophytes, definite JSN, possible bony deformity.

Grade 4: Large osteophytes, marked JSN, severe sclerosis, and definite bony deformity.
